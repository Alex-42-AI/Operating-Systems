Write a C program, that receives a path name, a command (executable file) and a list of parameters for it passed as parameters at the command line. It starts a descendant process and changes its image with the command. In the parent it waits for the descendant to finish and prints out its pid and exit code.
Write a C program, that receives 2 commands (without parameters) passed as parameters at the command line. The program implements the command1 && command2 construction.
Write a C program, that receives 2 commands (without parameters) passed as parameters at the command line. The program implements the command1 || command2 construction.
