Write a command procedure, that:
(LOGINFO) prints out properly formatted the current date, information on started sessions and a greeting.
(EXCHAN_NAMES) Switches the names of 2 existing regular files, passed as parameters at the command line.
Receives a user's name as a parameter at the command line and prints out how many times this user is logged in the system.
Receives a name of an existing text file and 2 numbers m < n as parameters at the command line. It prints out the file's lines, numbered n - m + 1 to n, sorted in reversed alphabetical order.
